This app was created to reduce the amount of
time to convert online media to a local audio
or video file - with a standalone tool.

Media created with this application is meant
for private/personal use.

[CURRENT UPDATES]

1.0.000.0004 / 2021.08.24

- Fixed issue with application crashing when pressing Go button

[PREVIOUS UPDATES]

1.0.000.0002 / 2021.08.23

- Fixed rare issue with application freezing halfway through first conversion

1.0.000.0001 / 2021.08.23

- Fixed issue with audio file conversions not working at times
- Changed audio formats to mp3, aac, m4a, flac, wav, and opus
- Changed video formats to mp4, webm, avi, mkv, mpg, and wmv
- Added process window for user to see status as conversions occur

1.0.000.0000 / 2021.08.23

- Initial Release of this tool
- Supports YT to Audio (mp3, aac, m4a, flac, wav, vorbis, and opus formats)
- Supports YT to Video (mp4 and webm formats)
- Added YT video name extractor
- Added CTGUpdater for automated updates
